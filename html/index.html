Experiencia

  • 2025
    Proyecto Futuro
    Tienda Online

    Mi primer proyecto este año fue este portafolio en formato de landing page, desarrollado con HTML, CSS y alojado en la web. El siguiente paso es una tienda online construida con tecnologías Full Stack:

    1. Frontend: HTML, CSS y JavaScript para la interfaz y experiencia de usuario.
    2. Backend: Node.js para la lógica del servidor.
    3. Base de datos: MongoDB para la gestión de productos, usuarios y pedidos.
    La tienda incluirá funciones clave como registro e inicio de sesión, carrito de compras, validación de datos, gestión de pedidos y seguridad en el manejo de la información.
